This project is a React form application built as part of the Week 1 assignment.

## Features
- Form validation for all required fields
- Error messages using conditional rendering
- Show / Hide password
- Phone number with country code validation
- PAN & Aadhaar validation
- Submit button disabled until form is valid
- Redirects to a success page on submit using React Router

## Tech Stack
- React (Vite)
- JavaScript
- React Router
